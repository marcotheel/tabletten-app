# Tabletten Erinnerung – PWA Version 6

Änderungen:
- PZN/Barcode-Feld wird nach Scan gefüllt.
- Anzeige heißt jetzt PZN/Barcode statt nur Code.
- Button „PZN-Daten suchen“ ergänzt.
